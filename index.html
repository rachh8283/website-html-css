Rachael Herman’s Portfolio Project

About the Project

For this trimester in Web Design & Development, I decided to go with option 1 in the portfolio pieces. Given my desire to continue with entrepreneurship following graduation, this five-page website will serve as a good demonstration of my HTML and CSS skills. It is a good marketing and portfolio piece to include on my current freelance website.

Elements Used

Required elements of text, headings, page layout, navigation, and images are included on every page. The style remains consistent by use of an external CSS file. Some specific elements used in this website include:

  • Lists
  • Tables
  • Boxes
  • Links
  • Widget

For CSS, I chose three major properties or concepts to use on the site:

  1. Positioning values
  2. Special effects
  3. CSS hover animation

The Branding

The colors, fonts, and logo are currently part of my freelance brand. While the website remains under construction, it is a work-in-progress that will help me move my career in a near direction.
